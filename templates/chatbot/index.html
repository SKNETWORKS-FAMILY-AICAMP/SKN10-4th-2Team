서비스 개요

AI 기반 다단계 평가 방식으로 와인 입문자를 위한 페어링, 용어 해석, 추천 제공

기획 배경
  • 🍇 와인 정보 복잡(라벨·품종·빈티지)
  • 🧀 실용적 페어링 정보 부족(한식·매운 음식)
  • 📱 홈술 및 비대면 큐레이션 수요 증가
UX 시나리오

질문 → 버튼 클릭 → sLLM → GPT-4o-mini 평가 → RAG → Tavily 보완 → 최종 응답

1. 서비스 개요

와인 초보자 맞춤형 챗봇으로 음식 페어링, 와인 용어 해석, 추천 정보를 AI 기반 다단계 평가 방식으로 제공합니다.

2. 기획 배경

  • 🍇 와인 정보의 복잡함: 라벨·품종·빈티지 등 미구조화 정보
  • 🧀 실용적 음식 페어링 정보 부족
  • 📱 홈술 트렌드 및 비대면 큐레이션 수요 증가

3. UX 시나리오

  1. 사용자 질문 입력(예: “삼겹살에 어울리는 와인 뭐가 있어요?”)
  2. ‘와인 추천 받기’ 버튼 클릭
  3. 챗봇 응답 프로세스:
    • 1차: sLLM 생성 → GPT-4o-mini 평가
    • 부족 시: FAISS RAG 검색 → 응답 보완
    • 추가 보완: Tavily 웹 요약 → 최종 응답

4. 기술 구성 요소

  • 🧠 LLM: Fine-tuned sLLM + GPT-4o-mini 평가
  • 🔍 RAG: FAISS 기반 검색, Tavily API
  • 💻 시스템: Django 백엔드, Bootstrap 프론트엔드

5. 데이터 소스

  • 내부 문서: 한국와인협회 가이드, 수입사 교육자료
  • 음식별 와인 페어링 표
  • 외부 콘텐츠: 유튜브 자막, 리뷰 블로그, 마트 콘텐츠

6. 사용자 케이스 예시

케이스 설명 챗봇 기능
🍛 음식 페어링 삼겹살, 매운탕, 치즈 추천 + 이유 제공
📘 용어 해설 까쇼, 빈티지, 바디감 용어 설명 + 예시 와인
🍷 구매 가이드 2~3만 원대 레드 와인 가격대별 추천 + 구매 팁
